# 💰 BudgetTracker

A personal finance web app for tracking daily expenses, setting spending limits, and monitoring monthly budgets — built with HTML, CSS, and JavaScript.

## Features

- **Log Expenses** — Submit spending entries with amount and category
- **Custom Categories** — Add and manage your own spending categories
- **Limits & Goals** — Set monthly spending limits and savings goals per category
- **Monthly Overview** — View a breakdown of spending for any selected month
- **USD ↔ LBP Converter** — Convert between US Dollar and Lebanese Pound using a live exchange rate
- **PDF Export** — Download a summary of any month's spending as a PDF file

## Tech Stack

- HTML, CSS, JavaScript
- Live exchange rate API for USD/LBP conversion
- PDF generation for monthly reports
